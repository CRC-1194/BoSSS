README - PARDISO
#==============#

The Pardiso Version used here as "PARDISO.dll", is the Pardiso distributed by the Intel MKL under the library files
"mkl_solver.lib mkl_intel_c.lib mkl_intel_thread.lib mkl_core.lib libiomp5md.lib".

Thus, according to https://software.intel.com/en-us/articles/which-intel-math-kernel-library-intel-mkl-libraries-are-redistributables
we are allowed to redistribute it.
