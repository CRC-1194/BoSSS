This note contains _minimal_ example configurations for several exemplary cases:

01-restart_by_fieldlist:
Shows how to restart a session using a list of fields.

02-restart_by_sessionid:
Shows how to restart a session using a session id.

03-formulas-with-heavyside:
Illustrates the usage of heavyside functions in formulas.

04-parameter-studies:
Illustrates the configuration of parameter studies which make it possible to
automatically run a BoSSS application for a whole bunch of parameter settings

05-queries:
Describes the usage of queries that can be used to extract quantities of
interest during BoSSS runs

06-query-mode:
Describes the query-mode which can be used to extract data from existing BoSSS
runs.
